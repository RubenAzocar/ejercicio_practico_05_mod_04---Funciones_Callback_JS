# Ejercicio Práctico - Funciones Callback en JavaScript

Este repositorio contiene la solución al ejercicio práctico sobre funciones callback en JavaScript, incluyendo validaciones, cálculos matemáticos y manejo de asincronía.

## Archivos principales

- **index.html**: Interfaz web para probar el ejercicio de callbacks.
- **ejercicio_callbacks.js**: Implementación de las funciones solicitadas.

## Instrucciones de uso

1. Abre `index.html` en tu navegador.
2. Haz clic en "Iniciar Ejercicio" y sigue las instrucciones en pantalla.
3. Los mensajes y resultados aparecerán en la página.

## Funcionalidades

- Validación de número ingresado por el usuario.
- Cálculo de sumatoria de números impares.
- Cálculo de sumatorias sucesivas y manejo de callbacks de éxito y error.
